# React E-commerce Storefront

A minimal React storefront demo built with Vite. Browse curated products, add items to a lightweight cart, and see instant updates across the layout.

## Getting started

```bash
npm install
npm run dev
```

## What is included

- Vite + React stack for fast local development
- Hard-coded product catalog with category filters
- Simple cart sidebar with quantity controls and total
- Responsive layout that works well on desktop and small screens
